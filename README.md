# Spelprodjekt


Projektet använder 
* [11ty](https://www.11ty.dev/) för att skapa html
* [Sass](https://sass-lang.com/) för att skapa css
* [Rollup](https://rollupjs.org/) för att kompilera javascript
* [Phaser 3](https://phaser.io/) som spelmotor
